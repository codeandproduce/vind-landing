Research

My research interests include: Natural Language Processing, Machine Learning, Deep Learning, GANs, making sense of the learning inside the black-box, and Hierarchal Temporal Memory

I am currently involved in A.I. Music research at NYU Shanghai with Dr. Gus Xia to create a generative model for music.

I've also conducted research as an entry to the Intel Science and Engineering Fair. The purpose of the project was to use NLP methods (Text Vectorization, N-Grams, TF-IDF, Linear SVMs optimized using both Coordinate & Gradient Descent) on Windows API System Calls to detect malware. Our model achieved a highest of 96% accuracy. The project won the 4th place Grand Award at Intel ISEF and special awards from numerous organizations (e.g. CAST, MAWHIBA).

The paper can be found here: (arXiv) NtMalDetect: A Machine Learning Approach to Malware Detection Using Native API System Calls

I self-studied Linear Algebra and Multivariable calculus on top of finishing AP Calculus BC at school and finished Andrew Ng's courses to make all this happen. I am now comfortable with much of the theoretical foundations as well as using TensorFlow and other Python modules.
